# 穿啥 - AI穿搭助手隐私政策

更新日期：2026 年 6 月 11 日

欢迎使用“穿啥 - AI穿搭助手”（以下简称“本应用”）。我们重视你的隐私和个人信息保护。本政策说明本应用如何处理你的信息，以及你在使用本应用时享有的相关权利。

## 1. 我们如何收集和使用信息

本应用用于提供 AI 穿搭评分、场景穿搭建议、配饰建议和穿搭历史记录等功能。为了实现这些功能，本应用可能会处理以下信息：

### 1.1 照片和相机信息

当你使用拍照或从相册选择图片功能时，本应用会访问你的相机或相册。你选择或拍摄的穿搭照片会用于生成 AI 穿搭分析、评分、配饰建议或相关穿搭内容。

本应用不会在未经你主动选择或拍摄的情况下读取你的照片。

### 1.2 你输入的文字内容

当你输入场景、风格偏好、穿搭需求或其他文字描述时，本应用会使用这些内容生成个性化穿搭建议。

### 1.3 AI 服务请求数据

为了提供 AI 分析和生成能力，本应用会将你主动提交的图片、文字描述和相关提示内容发送至第三方 AI 服务提供方进行处理。当前本应用使用的 AI 服务包括智谱 AI 相关接口。

这些数据仅用于完成你请求的 AI 分析、建议或图片生成结果。请避免上传或输入身份证件、银行卡、精确住址、联系方式、健康信息等敏感个人信息。

### 1.4 API Key

如果本应用要求你填写第三方 AI 服务 API Key，该 API Key 会保存在你的设备本地，用于向对应 AI 服务发起请求。本应用不会主动将你的 API Key 上传至开发者自有服务器。

### 1.5 穿搭历史记录

本应用会在你的设备本地保存穿搭历史记录，包括你保存的图片、缩略图、输入内容和 AI 返回结果，方便你后续查看。历史记录默认保存在本地设备中，你可以在应用内删除相关记录。

## 2. 我们不会收集的信息

除非后续版本另有明确说明，本应用目前不提供账号注册或登录功能，也不会主动收集以下信息：

- 姓名、身份证号等身份信息
- 精确地理位置
- 通讯录
- 支付信息
- 广告标识符
- 用于跨应用追踪的行为数据

## 3. 权限说明

本应用可能请求以下系统权限：

- 相机权限：用于拍摄穿搭照片并生成 AI 分析结果。
- 相册权限：用于选择已有穿搭照片并生成 AI 分析结果。
- 网络权限：用于连接第三方 AI 服务并返回分析或生成结果。

你可以在系统设置中管理或关闭相关权限。关闭权限后，部分功能可能无法正常使用。

## 4. 第三方服务

本应用可能调用第三方 AI 服务来完成图片理解、文字生成和图片生成等能力。你提交给 AI 功能的图片和文字内容可能会被发送至对应服务提供方处理。

第三方服务对数据的处理可能受其自身隐私政策和服务条款约束。建议你在使用相关功能前了解对应服务提供方的政策。

## 5. 数据存储与安全

本应用会采取合理措施保护本地保存的数据。请注意，任何互联网传输和电子存储方式都无法保证绝对安全。

如果你删除应用，保存在应用本地沙盒内的历史记录、图片缓存和 API Key 通常会随应用一并删除。

## 6. 儿童隐私

本应用不面向 13 周岁以下儿童。如果你是未成年人，应在监护人同意和指导下使用本应用。

## 7. 你的选择和权利

你可以通过以下方式管理自己的数据：

- 在应用内删除穿搭历史记录。
- 在系统设置中关闭相机或相册权限。
- 删除应用以清除本地保存的数据。
- 避免上传包含敏感个人信息的图片或文字。

## 8. 政策更新

我们可能会根据功能变化、法律法规要求或服务调整更新本隐私政策。更新后的政策会在本页面发布，并以页面顶部的更新日期为准。
